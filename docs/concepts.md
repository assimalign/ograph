# Interoperable Data Resolvers

One of the most useful benefits of GraphQL is the ability to resolve data from different data layers



```
├── Select
│   ├── Member 
│   │   ├── Member 
│   │   ├── Function (Scalar Only)
│   │   │   
│   ├── Function (Scalar Only) 
        └── Parameter
        │   │   │       ├── Member
        │   │   │       ├── Member
        │               └── Function
│   └── Value (Constant)
└── filter


```
