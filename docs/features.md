

# OGraph Features

- **OGraph Model Visualization:** a build in graph based visualization tool for both Web & desktop.
  - ***Graph Types:*** visualize the available services within the graph, see what events are invoked from interacting with those services and how they effect data layer.
    - *Service Graph*
    - *Event Graph*: get a graphical view of events that span off from interactions from the services layer
    - *Data Graph*
- **OGraph Model Stitching:** connect other OGraph services by defining fuzzy relationships to add to the data virtualization tooling.
